COMO FAZER DUAL BOOT

Dual boot é a prática de instalar dois ou mais sistemas operacionais (como Windows e Linux) em um único computador ou disco rígido. Ao ligar a máquina, uma tela de seleção permite que você escolha qual sistema deseja usar naquela sessão.



Elementos essencias para fazer um dual boot


PEN DRIVE

    Com estas configurações:
  • 8 ou mais GB de memória;
  • Totalmente limpo, sem nenhum arquivo;
    • Dica: Utilize um pen drive que não esteja utilizando.

RUFUS

O Rufus é um programa gratuito e de código aberto para Windows utilizado para formatar e criar unidades USB (como pen drives e cartões de memória) inicializáveis (bootáveis)

Baixe o Rufus no link abaixo:

Site oficial do Rufus

ISO DO SISTEMA OPERACIONAL

Será necessário a iso do sistema operacional que deseja baixar.

Se for o linux mint acesse o site logo abaixo:

Link para a página de download do linux Mint

COMO INSTALAR A ISO NO PEN DRIVE

-Primeiro: conecte o pen drive na sua máquina;

-Abra o Rufus;

-Veja se o pen drive está conectado na aba: DISPOSITIVO;

Selecione a ISO na aba: SELECIONAR;

Onde tem opção de imagem deixe: INSTALAÇÃO PADRÃO DO WINDOWS;

Na aba esquema de partição selecione: GPT;

Na aba sistema de destino selecione: UEFI NÃO CSM;

Onde tem esconder opções avançadas de formatação deixe selecionado: Formatação rápida e criar arquivos de nome estendido e ícone;

Após tudo isso clique em INICIAR e espere baixar.

Atenção!

Quando concluir o download não clique em iniciar novamente! Pode corromper o pen drive. Quando baixar feche a janela.

O QUE FAZER APÓS A INSTALAÇÃO

-Deixe o pen drive conectado;

-Desligue o pc;

-pesquise qual tecla abre a BIOS/UEFI do seu computador. É por modelo;

-ligue o computador novamente e já fique pressionado esta tecla para acessar a BIOS/UEFI;

-vai aparecer próximo do seu sistema operacional atual. clique nele e em seguida clique na primeira opção que aparecer.

-Isso deve abrir o novo sistema operacional.

BENEFÍCIOS DO DUAL BOOT

O dual boot permite instalar dois ou mais sistemas operacionais no mesmo computador, oferecendo a liberdade de escolher qual usar ao ligar a máquina. As maiores vantagens são a flexibilidade para rodar softwares exclusivos, o alto desempenho de hardware e a economia de espaço físico e dinheiro.
